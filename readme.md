### nvimtree

- `o` - 打开关闭文件夹，也可以打开文件
- `a` - 创建文件
- `r` - 重命名
- `x` - 剪切
- `c` - 拷贝
- `p` - 粘贴
- `d` - 删
- `f` - 搜索
